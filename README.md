# Linux Security Hardening Scripts

This repository contains practical Bash scripts for initial Linux server hardening.
Designed for Ubuntu / Debian-based systems and focused on production security best practices.

## Covered areas
- SSH hardening (key-only auth, root disable, 2FA)
- Firewall configuration (UFW)
- User and privilege management
- Fail2Ban brute-force protection
- Basic audit logging

## Use cases
- Initial VPS setup
- Production server hardening
- Security-focused CI/CD environments
- Compliance-oriented deployments

⚠️ Scripts should be reviewed and tested in a staging environment before production use.
